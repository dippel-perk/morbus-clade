Hilfe zur Demo

Die Website ist aktuell noch eine Demo, die aber die einfache Realisierbarkeit des Konzeptes in der Realität veranschaulichen soll. Auf dieser Seite wirst du einmal durch die Demo geführt.

  • Testperson
  • Gesundheitsamt

1. Testanmeldung

Begib dich zunächst über die Startseite auf die Testanmeldungs-Seite.

2. Formular ausfüllen

Fülle nun das Formular für den Test aus und registriere dich.

3. Testsimulation

Auf der folgenden Seite befinden sich deine persönlichen Informationen und ein Qr Code. Nun würdest du zur nächstgelegenen Teststation fahren und dort den QR-Code beim Testen vorzeigen. 4. Warten auf das Ergebnis

Nach Abgeben der Probe an der Teststation muss man nun auf das Ergebnis warten. Auf der Website kann man sehen an welcher Teststation die Probe abgegeben wurde und welches Testlabor die Probe auswertet. Wenn das Testlabor die Probe getestet hat, tragen sie das Ergebnis ein. Dies kann in der Demo oben mit einem Klick auf das jeweilige Testergebnis simuliert werden.

5. Kontaktpersonen hinzufügen

Wenn das Testergebnis positiv war, bekommt die Testperson einen Link per E-Mail zugeschickt. Durch Aufrufen des Links öffnet sich eine Seite auf der Kontaktpersonen hinzugefügt werden können. Diese Seite wird in der Demo automatisch geöffnet nachdem man das Testergebnis als "Positiv" ausgewählt hat.

6. Daten der Kontaktperson angeben

Beim Hinzufügen der Kontakperson werden die Daten der jeweiligen Person angegeben, die Intensität des Kontaktes und es kann eine kleine Beschreibung hinzugefügt werden, die den Kontakt beschreibt.

1. Login

Um auf die Verwaltungsseite eines jeweiligen Gesundheitsamtes zuzugreifen, muss man sich zunächst einloggen. Um dies zu testen kann man sich mit dem Demo-Account "gesundheitsamt-braunschweig" un dem Passwort "test" einloggen.

2. Übersicht

Nach dem Login gibt es für das Gesundheitsamt eine Übersicht über alle getesteten Personen. Diese Liste kann nach bestimmten Kriterien gefiltert werden. In Zukunft wird es auch noch möglich sein aggregierte Statistiken über die einzelnen Testpersonen einzusehen. Durch Klicken auf eine Testperson gelangt man zur Detailansicht.

3. Detailansicht

In der Detailansicht können die genauen Daten zu einer Testperson und dessen Kontaktpersonen eingesehen werden.
